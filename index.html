Connect Team Tracker
Enter your Supabase project credentials below. Data is stored directly in your own Supabase database and syncs across every device.
1
Create a free Supabase project
Go to supabase.com → New project → pick a name and strong password → choose a region close to you → click Create new project. Wait ~1 min for it to spin up.
2
Run the schema SQL
In your project → SQL Editor → New query → paste the contents of schema.sql (included in this project folder) → click Run. This creates the 4 tables.
3
Get your credentials
Go to Project Settings → API. Copy the Project URL and the anon / public key into the fields below.
Supabase Project URL
Supabase Anon / Public Key
